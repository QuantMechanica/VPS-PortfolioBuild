This evidence document details the execution of the exit surgery task for **QM5_12821** (T-WIN Currency-Strength Meter Basket strategy) on 2026-07-02. 

The surgery isolates the three exit-amputation mechanisms that collapse the hold time and kill the core edge (from the full-run 2018-2024 backtest evidence):
1.  **Moments of decay exit:** `strategy_close_on_signal_decay` closes the basket when `EvaluateSignal` expires (instead of only on a true strength flip).
2.  **Daily session flat:** `strategy_flat_hhmm` (21:00) kills multi-day holdings.
3.  **Pending order lifetime limit:** `strategy_pending_expiration_minutes` (60m) expires unfilled leg limits, leading to single-leg runs instead of full-basket diversification.
