# XNG March Transseason Short

Canonical card:
`strategy-seeds/cards/approved/QM5_12894_xng-mar-transseason-short_card.md`.

This EA implements `QM5_12894` as a D1 `XNGUSD.DWX` March-to-mid-April
transseason short. It uses only MT5 OHLC, broker calendar, spread, SMA, ATR,
and V5 framework state at runtime. It does not read EIA, storage, weather,
power-load, futures-curve, CSV, API, forecast, or ML data.
